Баг-репорт №1: 
Функция capitilize изменяет регистр первых букв во всех словах в стороке.
Шаги:
В ходе автотеста  применить функцию capitilize к строке "nothing 123 Nothing"
Ожидаемый результат:
Функция делает первую букву заглавной и возвращает этот же текст, "Nothing 123 Nothing"
Фактический результат: Функция меняет на противоположный регистр всех первых букв слов в строке, "Nothing 123 nothing"
Ответ pytest: FAILED test_string_utils.py::test_capitilize[nothing 123 Nothing-Nothing 123 Nothing] - AssertionError: assert 'Nothing 123 nothing' == 'Nothing 123 Nothing'

Баг-репорт №2:
Функция starts_with возвращает значение True, при применении к непустой строке для обработки пустого искомого символа.
Шаги:
В ходе автотеста задать параметры "nothing", "" в функции starts_with
Ожидаемый результат:
Функция вернёт "False", т.к. непустая строка не может начиться с пустого символа
Фактический результат:
Функция возвращает параметр "True"
Ответ pytest:FAILED test_string_utils.py::test_starts_with[nothing--False] - assert True == False
